Installation

Unzip the downloaded files to the local disk on your computer. Inside the extracted folder there will be a folder with documentation and a folder "assets" containing all source files. Upload the source files to your server via FTP.

What's Included

  • assets
      • cssContaining all css files
      • imagesHere is a place to put your media files like logo, images, etc.
      • jsContaining all js files
      • sassContaining all scss files
      • index.html Main HTML file
      • contact-mail.php Contact Form Submission PHP file
      • newsletter.php Newsletter Form Submission PHP file
      • .htaccess Configuration file
      • favicon.icon Favicon icon

Components

Preloader

Page Loading Transitions are enabled by default. If you wish to disable the page loading transition you can simply delete below section.

 <div class="site-loader">
   [Loader Content]
 </div>

Favicon

Favicon is an icon associated with URL that is displayed at various places such as in a browser’s address bar or next to the site name in a bookmark list. Following code is for favicon, to change just place your favicon.ico file in root path

<link rel="shortcut icon" href="favicon.ico" type="image/x-icon">

Logo

To change logo, place your logo file in images folder and find navbar-brand class in index.html file. Replace the current logo image name with yours.

    <a class="navbar-brand" href="index.html">
        <img src="images/logo.png" alt="logo image" class="img-fluid"/>
    </a>

Navigation

Follwing section is for the Navbar menu, Find class .nav-link inside nav and make required changes in Navbar

    <nav class="navbar navbar-expand-lg navbar-togglable navbar-dark">
        [Navbar Content]
    </nav>

Basic Styling

Color Scheme

Bitency template come up with 7 preset color schemes. CSS files are separated based of color scheme, so you can easily use them.

Bitency template uses sass variable, so changing color scheme is much easier. You just need to add your color code at one place and it will apply everywhere.

To add your own color scheme, go to template-color.scss file inside css folder and change PrimaryColor, SecondaryColor, AlternateColor vairables color.

Background Image / Icons

Bitency template added background images path in style.scss background css rules. and Icons path are added in index.html file.

To change background image or icons go to the respective section and change the image/icon name.

Please make sure your new images/icons is placed in images folder

Font

Bitency template uses Google WebFont, there are thousands of fonts available, you can select fonts as per your need from google fonts library

Find the current added font link inside theindex.html file

<link href="https://fonts.googleapis.com/css?family=Poppins:300,300i,400,400i,500,600,700,800,900%7CWork+Sans:300,400,500" rel="stylesheet">

In Order to change the fonts, you neeed to change the above link with your new google fonts link. and change font-family in style.scss in sass folder

You can learn more about how to use google fonts here

Buttons

Bitency template uses .btn-alpha class name for define button, In order to change the button style, go to style.scss inside sass folder and make required changes.

Count Down

Bitency ICO template use jquery basic count down.

You can easily change the ICO date, you just need to go to main.js file in js folder and find the following code

/*---- Add you ICO date and time here---*/
var endTime = new Date("October 20, 2018 18:00:00 PDT");
var endTime = (Date.parse(endTime)) / 1000;
var now = new Date();
var now = (Date.parse(now) / 1000);
var timeLeft = endTime - now;
var days = Math.floor(timeLeft / 86400);
var hours = Math.floor((timeLeft - (days * 86400)) / 3600);
var minutes = Math.floor((timeLeft - (days * 86400) - (hours * 3600 )) / 60);
var seconds = Math.floor((timeLeft - (days * 86400) - (hours * 3600) - (minutes * 60)));
if (days < "10") { days = "0" + days; }
if (hours < "10") { hours = "0" + hours; }
if (minutes < "10") { minutes = "0" + minutes; }
if (seconds < "10") { seconds = "0" + seconds; }
$(".days").html(days + "Days");
$(".hours").html(hours + "Hours");
$(".minutes").html(minutes + "Minutes");
$(".seconds").html(seconds + "Seconds");
}

Newsletter form

To connect the newsletter subscribe form to your email account please enter your email address into the newsletter.phpfile under :$recipient = "youremail@gmail.com";

Additionally you are able to change the subject line of the email you will get after sending a message for this please use the php line: $subject = "Your Subject $email";
